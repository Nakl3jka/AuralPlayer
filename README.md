# 🎵 AuralPlayer

AuralPlayer is a free, Spotify-like music streaming platform that allows users to upload, discover, and listen to music online. It provides a modern, responsive interface designed for artists and listeners alike.

## ✨ Features

- 🎧 Stream music online
- ⬆️ Upload your own tracks
- 👤 User accounts and profiles
- 🎤 Artist profiles
- 💿 Albums and singles
- 📃 Create and manage playlists
- ❤️ Like favorite songs
- 🔍 Search for songs, artists, and albums
- 📱 Responsive design for desktop and mobile
- 🌙 Dark mode support
- ⚡ Fast and lightweight
